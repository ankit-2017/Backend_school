How to integrate bootstrap in Django

There are two ways to use bootstrap in web page

Bootstrap CDN (Content Delivery Network) CDN contains all bootstrap libraries online, to access these libraries you will need an internet connection while accessing for first time, then browser stores all libraries in cache memory and libraries could be accessed offline.Add CDN in the head section of HTML page.

Download Bootstrap library , extract downloaded folder , you will get css, js and icon folder ,select minimised libraries like bootstrap.min.css. for javascript file select bootstrap.min.js. These libraries are compressed version that reduces white spaces, comments etc. from code so these are fast to load.



This is forms.py with bootstrap class

from django import forms
from .models import *

class student_form(forms.ModelForm):
    name = forms.CharField(widget=forms.TextInput(attrs={'class':'form-control',
        'placeholder':'Enter your name'}),required=True, max_length=100)

    emailid = forms.CharField(widget=forms.EmailInput(attrs={'class':'form-control',
        'placeholder':'Enter your Email'}),required=True, max_length=100)

    city = forms.CharField(widget=forms.TextInput(attrs={'class':'form-control',
        'placeholder':'Enter your city'}), required=False, max_length=100)

    marks = forms.CharField(widget=forms.NumberInput(attrs={'class':'form-control',
        'placeholder':'Enter your marks'}), required=True, max_length=10)

    class Meta():
        model = student
        fields = ['name','emailid','city','marks']
