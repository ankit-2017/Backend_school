Django installation on linux distribution

Install Django using pip

Python 2x and 3x both is pre-installed in linux distributions, to check version python 2.7 just type python in terminal(open terminal by using Ctrl + Alt + T. Now update system and install pip(python package manager) by using following command

For python 3x version

root@mypc:~$ sudo apt-get update
root@mypc:~$ sudo apt-get install python3-pip python3-dev

For python 2x version

root@mypc:~$ sudo apt-get install python-pip python-dev

For python 3x versions


Run the following command in your terminal. 

root@mypc:~$ sudo pip3 install django==1.9

Specify which version of django you want to install.
If you will not specify any version then it will automatically install the latest version of Django.To install with latest version of django run following command in cmd. 


root@mypc:~$ sudo pip3 install django

For Python 2x version

Run the following command in cmd(terminal) 

root@mypc:~$ sudo pip install django==1.9
                or
root@mypc:~$ sudo pip install django

After installing django check whether it is installed correctly or not! Open cmd and type python then write following given command:


import django
django.__version__
'1.11.7'

If you will get the version of django then it is successfully installed on your system.

Create Project


To start django project,open command prompt(cmd), create a directory where you want to keep django project. Then change directory to new created directory.then run following commands in cmd: 

root@PC:~\pathToDirectory\myProject$ django-admin startproject myProject
root@PC:~\pathToDirectory\myProject$ django-admin startapp polls

root@PC:~\pathToDirectory\myProject$ python3 manage.py migrate
To run django server, must ensure you are in directory where manage.py file is present, then run following command: 
root@PC:~\pathToDirectory\myProject$ python3 manage.py runserver
Add app (polls) to installed apps in settings.py file which is present in project directory. 
        settings.py
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'polls',
]
Open any browser and enter following URL localhost:8000
If following message display on your browser then you have done it.
