Install Django using pip

For python 3x versions


Run the following command in your cmd(terminal). 

C:\pathToDirectory\myProject> pip3 install django==1.9

Specify which version of django you want to install.
If you will not specify any version then it will automatically install the latest version of Django.To install with latest version of django run following command in cmd. 


C:\pathToDirectory\myProject> pip3 install django

For Python 2x version

Run the following command in cmd(terminal) 

C:\pathToDirectory\myProject> pip install django==1.9
                or
C:\pathToDirectory\myProject> pip install django

After installing django check whether it is installed correctly or not! Open cmd and type python then write following given command:


import django
django.__version__
'1.11.7'

If you will get the version of django then it is successfully installed on your system.

Create Project


To start django project,open command prompt(cmd), create a directory where you want to keep django project. Then change directory to new created directory.then run following commands in cmd: 

C:\pathToDirectory\myProject> django-admin startproject myProject
C:\pathToDirectory\myProject> django-admin startapp polls

C:\pathToDirectory\myProject> manage.py migratemigrate database in project
To run django server, must ensure you are in directory where manage.py file is present, then run following command: 
C:\pathToDirectory\myProject> manage.py runserver
Add app (polls) to installed apps in settings.py file which is present in project directory. 
        settings.py
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'polls', Your app
]
Open any browser and enter following URL localhost:8000
If following message display on your browser then you have done it.
